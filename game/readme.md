# Visualization Your GPTWorld on Browser!

## Setup

essentially, in your command line

```
cd text_grid
npm install
npm install phaser
npm run dev
```

If you have any question, refer to [this blog](https://saricden.com/how-to-setup-a-phaser-3-project-with-vite)
